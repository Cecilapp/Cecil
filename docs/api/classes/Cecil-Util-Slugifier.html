Slugifier

Slugifier.php : 26

Converts arbitrary paths/strings into URI-safe slugs.

Preserves '.', '_', and '/' characters, handles non-ASCII (including CJK) via the Symfony AsciiSlugger, and replaces any remaining unsafe characters with dashes.
